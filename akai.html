01
Wat is AKAI

Een AI die écht van mij is.

De meeste AI assistenten — Siri, ChatGPT, Google Assistant — draaien op servers van grote bedrijven. Jouw gesprekken, jouw vragen, jouw data: allemaal opgeslagen ergens in een datacenter waar je geen controle over hebt.

AKAI is anders. Hij draait op de Raspberry Pi 5 die bij mij thuis staat. Elk gesprek blijft lokaal. Elke tool die hij aanroept draait op mijn eigen server. En toch is hij net zo bereikbaar als elke andere chatbot — gewoon via Telegram, dag en nacht.

AKAI kent mij. Hij weet hoe ik heet, wat mijn projecten zijn, welke AI modellen ik gebruik en hoe mijn trading strategie eruitziet. Die context staat in bestanden op Nexus — niet in een cloud database van een derde partij.

03
Hoe het werkt

Van bericht tot antwoord.

STAP 01
Kanaal ontvangt bericht

Een bericht komt binnen via Telegram of een telefoongesprek via Twilio. De OpenClaw gateway luistert op poort 18789 en ontvangt het event via een webhook of long-polling connectie.

Telegram Bot API Twilio Voice Webhook OpenClaw Gateway :18789
STAP 02
Context laden

AKAI laadt zijn identiteit en geheugen: SOUL.md (karakter), IDENTITY.md (wie hij is), USER.md (wie ik ben), MEMORY.md (wat hij onthouden heeft), TOOLS.md (wat hij kan doen). Dit vormt de system prompt.

SOUL.md IDENTITY.md USER.md MEMORY.md TOOLS.md
STAP 03
Model selectie & denken

De gateway selecteert het juiste AI model op basis van de taak. Standaard gesprekken gaan naar Step 3.5 Flash via OpenRouter (razend snel, vrijwel gratis). Complexe taken of code naar Claude via de API. Maximale privacy naar lokale Ollama modellen.

Step 3.5 Flash (default) Claude Sonnet (complex) Ollama lokaal (privacy)
STAP 04
Tools uitvoeren

Als AKAI een tool wil gebruiken — bijv. kalender checken, trade loggen of server status ophalen — voert hij die direct uit op Nexus via scripts of API calls naar lokale services. Geen externe afhankelijkheid.

gog (Google Calendar) Nexus scripts Trading Journal API Bash tools
STAP 05
Antwoord & geheugen

AKAI stuurt het antwoord terug via het kanaal (Telegram bericht of gesproken stem via Twilio). Als er iets is om te onthouden, schrijft hij dat weg naar MEMORY.md voor toekomstige gesprekken.

Telegram reply Twilio TTS MEMORY.md update

Technisch

Hoe AKAI zichzelf omschrijft.

De kern van AKAI's identiteit staat in tekstbestanden op de server. Dit is een fragment uit SOUL.md — de file die bepaalt wie AKAI is en hoe hij denkt.

~/.openclaw/workspace/SOUL.md
# AKAI — Kern identiteit Naam: AKAI Platform: OpenClaw Gateway op Nexus (Raspberry Pi 5) Eigenaar: Alano Kricha, Voorburg # Karakter Ik ben direct, eerlijk en efficient. Ik zeg wat ik denk zonder onnodige woorden. Ik ken Alano's projecten, doelen en voorkeuren. # Kernwaarden Privacy: Alle data blijft op Nexus. Geen cloud. Geen tracking. Controle: Alano heeft volledige controle over mij en mijn config. Nut: Ik bespaar tijd. Elke interactie moet waarde toevoegen. # Wat ik weet — Nexus infrastructuur (17 services, alles draaiend) — Trading Journal (XAUUSD strategie, psychologie tracking) — Alano's kalender, notities en projecten — Hoe ik tools moet aanroepen en scripts moet uitvoeren

[Unit] Description=AKAI — OpenClaw Gateway After=network.target [Service] User=ak ExecStart=/home/ak/.npm-global/bin/openclaw gateway --port 18789 Restart=always RestartSec=5 EnvironmentFile=/home/ak/.openclaw/gateway.env [Install] WantedBy=multi-user.target
06
Interessante keuzes

Wat ik heb geleerd.

Context is exponentieel duur. Elke API call stuurt de volledige gespreksgeschiedenis mee. Na 6 dagen zonder reset had één gesprek 40 miljoen tokens — maar Step 3.5 Flash is zo goedkoop dat dit maar $1.33 kostte.

Identiteitsbestanden zijn krachtig. SOUL.md, MEMORY.md en USER.md zijn geen technisch trucje — ze zijn wat AKAI maakt tot een assistent die écht nuttig is. Zonder die context is het gewoon een generieke chatbot.

Nooit doctor --fix uitvoeren. OpenClaw heeft een commando dat zichzelf zou moeten herstellen maar in de praktijk de hele configuratie wist. Twee keer geleerd: 7 april 2026 en al eerder op 2 april.

Meerdere modellen zijn beter dan één. Door verschillende modellen te kunnen inzetten voor verschillende taken bespaar ik geld, verhoog ik privacy en haal ik het beste uit elke situatie. Het beste model wisselt ook nog elk kwartaal.

Self-hosted AI is rijp voor productiegebruik. 24/7 uptime, sub-seconde Telegram responses, voice calls via Twilio — alles draait stabiel op een Pi die 5 euro stroom per maand kost. De technologie is er.

07
Het platform — OpenClaw

Gebouwd op OpenClaw.
Ooit "MoltBot" geheten.

AKAI draait op OpenClaw — een open-source AI gateway die ooit begon als "MoltBot", een persoonlijk project van Peter Steinberger voor zijn eigen AI assistent "Molty" 🦞, een ruimtekreeft. Het groeide uit tot een volledig platform, gesponsord door OpenAI, GitHub en NVIDIA.

Het is geen ChatGPT wrapper. Het is een volledig persoonlijk AI besturingssysteem — met channels, canvas, tools, cron jobs, agents en meer. De mogelijkheden zijn ronduit absurd.

Vraag AKAI om een feature te bouwen en hij spawnt Claude Code als achtergrond agent op Nexus. Hij browset bestanden, schrijft code, runt tests — terwijl jij iets anders doet. Het resultaat komt terug als Telegram bericht.

Met OpenClaw als Android node kan AKAI je telefoon lezen en aansturen: notificaties ophalen, locatie checken, SMS sturen, foto's bekijken, contacten opzoeken, apps updaten. Jij typt een bericht — hij pakt je telefoon.

AKAI kan een volledige Chrome browser aansturen: pagina's bezoeken, formulieren invullen, screenshots maken, data scrapen. "Kijk wat de prijs van X is" — hij opent de site, leest het en rapporteert terug.

Geplande taken zonder cron te schrijven. "Stuur me elke ochtend om 8:00 het weerbericht + mijn agenda van vandaag" — AKAI regelt het zelf via een ingebouwd cron systeem. Geen terminal, geen crontab.

self-improving skill SOUL.md writes Skill vetting
Scenario — The Lazy Developer

"Hey AKAI, voeg een dark mode toggle toe aan mijn portfolio." — AKAI spawnt Claude Code op Nexus, vindt de CSS, voegt de toggle toe, runt de tests en stuurt je een Telegram bericht terug: "Done. Gecommit naar main."

Scenario — The Morning Brief

Elke ochtend om 8:00: AKAI stuurt vanzelf een Telegram bericht met het weer in Voorburg, jouw agenda voor vandaag en de Nexus server status. Geen actie van jou nodig — een cron job regelt het.

Scenario — The Living Dashboard

"Maak een live dashboard van mijn trades van vandaag." — AKAI bouwt een HTML/CSS chart, vraagt data op uit de Trading Journal en pusht het als interactieve pagina direct naar je iPhone scherm. Geen app, geen link — gewoon: boom, daar is het.

Scenario — The Twitch Mod

AKAI kan ook als chatbot in Twitch livestreams draaien. Kijkers stellen vragen, AKAI beantwoordt ze in de chat — met toegang tot alle tools. Een streamer kan AKAI live code laten schrijven terwijl het publiek meekijkt.

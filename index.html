AKAI — Persoonlijke AI Assistent

Self-hosted AI die praat, denkt en handelt

Raspberry Pi 5 Python Telegram Twilio Ollama Claude API

AKAI is mijn persoonlijke AI assistent die draait op mijn eigen Raspberry Pi server — geen abonnement, geen cloud, geen data-deling. Hij reageert op Telegram berichten, neemt gesprekken aan via mijn telefoon en heeft toegang tot tools zoals mijn kalender en notities. AKAI kent mijn context en werkt 24/7.

  • Reageert op Telegram berichten — dag en nacht
  • Voicegespreken via Twilio, doorgesteld naar mijn nummer
  • Toegang tot tools: kalender, systeem, notities, Trading Journal
  • 100% self-hosted — draait lokaal op Nexus

Nexus — Raspberry Pi 5 Server

24/7 self-hosted infrastructuur voor AI en automation

Raspberry Pi 5 Docker Ollama Pi-hole Tailscale UFW

Nexus is de Raspberry Pi 5 die bij mij thuis staat en alles draait. AI modellen, media server, DNS filtering, smart home, monitoring — allemaal self-hosted en altijd online. Beveiligd via zero-trust networking met Tailscale VPN en een strikte firewall.

  • 17 actieve services: Jellyfin, Grafana, Prometheus, Homebridge en meer
  • Pi-hole blokkeert advertenties en tracking voor het hele netwerk
  • Zero-trust beveiliging met Tailscale VPN en UFW firewall
  • Apple HomeKit integratie via Homebridge
